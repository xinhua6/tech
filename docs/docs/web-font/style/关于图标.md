# 关于图标
图标的使用：尽量使用自己项目中所使用的组件框架中的自带icon，如果没有，可以有如下方法解决：
1. 直接下载svg图标替代，可以在编辑器中打开图片修改width和height，fill便可以直接更改大小和颜色
2. 使用iconfont，在项目中导入在线连接，直接使用font-class。缺点：如果存在查漏，增加后链接会更新，不利于后期维护
3. 纯css制作图标，参考张鑫旭的css图标库，适用于简单的图标
